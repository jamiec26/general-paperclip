# Contributing

## General Workflow

- One Paperclip issue → one branch → one PR.
- Branch naming tied to the issue identifier (e.g. `afr-NN-short-slug`).
- PR body links back to its Paperclip issue ("Closes AFR-NN"); the Paperclip issue links to the PR.
- Secrets live in GitHub Actions secrets / env, never in commits or Paperclip threads.
